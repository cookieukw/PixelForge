# Pixel Forge
<div align="center">
<img src="https://raw.githubusercontent.com/cookieukw/PixelForge/main/public/sun.png" width="256" height="256">
</div>

<br>

Pixel Forge is a web application for creating canvas games using the Blockly library. With Pixel Forge, you can create your own games and export them as HTML files. The application is built using Vite, React, TypeScript, and Redux.

<br>

# Features

Feature|Description
-|-
Create canvas game|Create canvas games with ease using Blockly
Export as HTML|Export your game as an HTML file
Easy to use|Easy to use interface
Tech stack|Built with Vite, React, TypeScript, and Redux

# Installation
Clone the repository

```bash
git clone https://github.com/cookieukw/PixelForge.git
```

Navigate to the project directory

```bash
cd PixelForge
```

Install dependencies

```bash
npm i
```

Start the application

```bash
npm run dev
```

# Usage

- Open your browser and navigate to ```http://localhost:3000```.
- Use the Blockly library to create your own game
- Export your game as an HTML file
